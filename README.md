# 🌿 Greenden – Plant Store Website

Greenden is a simple, responsive plant store website built using **HTML** and **Tailwind CSS**.  
The project focuses on clean UI design, responsiveness, and modern styling using utility-first CSS.

---

## 🚀 Features

- 🌱 Modern and clean UI design  
- 📱 Fully responsive (Mobile, Tablet & Desktop)  
- 🎨 Styled using Tailwind CSS  
- 🧱 Pure HTML structure (No frameworks)  
- ⚡ Fast and lightweight  

---

## 🛠️ Technologies Used

- **HTML5**
- **Tailwind CSS**
